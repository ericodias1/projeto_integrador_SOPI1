Um sistema operacional procura sempre deixar a utilização eficiente a favor do usuário, gerenciando o hardware e deixando o usuário utilizando apenas uma interface gráfica para utilizar a máquina. [Machado and Maia, 2007]

Durante o artigo o Android será citado como sistema operacional, plataforma ou tecnologia. A diferença nesses 3 termos está apenas no modo da utilização do Android. Quando usado como sistema operacional, podemos imaginar o usuário final, aquele que utiliza a interface para tarefas triviais, ou seja, não se preocupa com nenhuma parte do desenvolvimento ou back-end(parte do sistema que não é visível ao usuário final). O Android como plataforma é usado como uma base para desenvolvimento de aplicativos, ou seja, quando o programador utiliza o Android como um padrão de desenvolvimento para seus aplicativos, sempre pensando na arquitetura do sistema ao criar cada aplicação. Por fim, o Android como tecnologia é o que estamos mais acostumados a ler, ver e ouvir. Tecnologia porque trouxe aos seus usuários coisas que até então eram praticamente impossíveis de serem imaginadas, como por exemplo, utilizar um aparelho celular para ligar uma luz da sua casa.[Pereira, Lúcio and Da Silva, Michel, 2009]

O Android é um sistema operacional para dispositivos móveis criado pela OHA (na qual pertence uma das maiores empresas de software, a Google). Ele é open-source, ou seja, qualquer usuário tem acesso a seu código e pode alterá-lo para uso próprio. [Andrade, 2011]
