Utilizou-se CSS na página para a formatação dos textos, menus e imagens.

Foi utilizada uma biblioteca CSS chamada Bootstrap (getbootstrap.com). Nela contém elementos pré-formatados como menus, tabelas, listas, botões, etc. Esta biblioteca também disponibiliza funções de responsividade, ou seja, ao utilizar o sistema de Grids (que consiste em criar grids para disponibilizar a página) automaticamente a página é bem visualizada em qualquer tamanho de navegador.
